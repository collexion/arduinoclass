Arduino Class

These pages contain the materials for our Arduino workshop at the Living Arts and Science Center from July 15-19, 2013. Each class is a three hour session.

Contributors

Lots of the initial materials were provided by Paul for the intro to engineering class he did at UK. The teachers have since revised the materials.

Contributing

These pages are generated through a git repository. Check out the repository and switch to the gh-pages branch to see the contents (in the Markdown syntax). Make a pull request if you want to suggest updates to the material. Ask on the mailing list collexion@googlegroups.com if any of those previous sentences were confusing.
